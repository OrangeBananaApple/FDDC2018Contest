# FDDC2018Contest
## HTML Data Parser
```
usage: html_data_parser.py [-h] [--mode [{single,batch}]] input_path output_path

positional arguments:
  input_path            path of input file/directory
  output_path           path of output file/directory

optional arguments:
  -h, --help              show this help message and exit
  --mode [{single,batch}] which mode should this program works in

```
注意：
1. 在single模式下，input_path/output_path必须都是文件路径
2. 在batch模式下，input_path/output_path必须都是文件夹路径
